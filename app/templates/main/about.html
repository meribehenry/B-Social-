About

B-Social is an experimental project created by Meribe Henry to practice web development skills. It only includes the basics for now, as I'm still learning and on the journey to become a better developer.

This application allows you to share your content with others and get feedback from them. It still has many features to be implemented and with time, it will become a more robust platform. So as of now, it's a simple space for sharing and connecting with others. You can create post, comment on post, edit post, follow a particular creator and react on their posts. More update would soon be made so stay tuned for more updates!

If you have any questions or feedback, feel free to reach out to me on Twitter or GitHub. You can also just give your feedback directly in the application and also see other user feedbacks.

Thank you for visiting B-Social and I hope you enjoy using it!
